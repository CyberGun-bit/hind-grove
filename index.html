📧 Email reminder settings
How automated cleaning reminders work:
This app uses EmailJS (free — 200 emails/month) to automatically send a Gmail reminder to each person 1 day before their cleaning turn.

Setup steps:
1. Go to emailjs.com → create a free account
2. Add a service → choose Gmail → connect your Gmail → copy the Service ID
3. Create an email template → use variables: {{to_name}}, {{to_email}}, {{cleaning_date}}, {{tasks}} → copy the Template ID
4. Go to Account → copy your Public Key
5. Paste all three below and click Save
