✅ Automatic email reminders are handled by GitHub Actions
Every day at 9:00 AM London time, GitHub automatically checks the rota and sends a reminder email to whoever cleans the next day. Each person only ever gets one email per turn — no duplicates. No one needs to open the website for this to work!

To activate: add your 3 EmailJS keys as GitHub Secrets (see instructions below).
🔐 GitHub Secrets setup (one time only)
Add these 3 secrets to your GitHub repo:

1. Go to github.com/CyberGun-bit/hind-grove
2. Click Settings → Secrets and variables → Actions
3. Click "New repository secret" and add these one by one:

  • Name: EMAILJS_PUBLIC_KEY → Value: your EmailJS public key
  • Name: EMAILJS_SERVICE_ID → Value: service_mseqn1e
  • Name: EMAILJS_TEMPLATE_ID → Value: your template ID

4. To get your Public Key: EmailJS dashboard → Account → API Keys
5. To get your Template ID: EmailJS dashboard → Email Templates → your template
